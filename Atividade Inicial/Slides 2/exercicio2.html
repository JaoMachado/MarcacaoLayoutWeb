Pagina Usando os Elementos


Exemplo Tag: "hr"

A tag "hr" define uma quebra temática em uma página HTML (por exemplo, uma mudança de tópico). É mais frequentemente exibido como uma régua horizontal usada para separar conteúdo (ou definir uma alteração) em uma página HTML.


Exemplo Tag: "em"

A tag "em" é usada para definir texto enfatizado. O conteúdo dentro é normalmente exibido em itálico. Um leitor de tela pronunciará as palavras dentro da tag com ênfase, usando a ênfase verbal.


Exemplo Tag: "small"

A tag "small" define textos menores (como direitos autorais e outros comentários paralelos).
Dica: Esta tag não está obsoleta, mas é possível obter um efeito mais rico (ou o mesmo) com CSS.


Exemplo Tag: "mark"

A tag "mark" define o texto que deve ser marcado ou destacado.


Exemplo Tag: "del"

A tag "del" define o texto que foi excluído de um documento. Os navegadores geralmente riscam o texto excluído.


Exemplo Tag: "ins"

A tag "ins" define um texto que foi inserido em um documento. Os navegadores geralmente sublinham o texto inserido.


Exemplo Tag: "sub"

A tag "sub" define texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H2O ou C6H12O6.


Exemplo Tag: "sup"

A tag "sup" define texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e às vezes é renderizado em uma fonte menor. O texto sobrescrito pode ser usado para notas de rodapé, como WWW[1] ou ax4 + bx3 + cx2 + dx + e = 0.


Exemplo Tag: "q"

A tag "q" define uma citação curta. Os navegadores normalmente inserem aspas ao redor da citação.


Exemplo Tag: "code"

A tag "code" é usada para definir um pedaço de código de computador. O conteúdo dentro é exibido na fonte monoespaçada padrão do navegador.
